Elementos block

Os elementos block como titúlos, parágrafos eu tabelas, tem a largura de toda a tela. Outro comportamento do block é que ele sempre vai ficar um abaixo do outro.

Elementos inline

Os elementos inline como links, spans e imagens são diferentes, sua largura depende de seu conteúdo e eles ficam sempre um a frente do outro(lado).

Elementos inline-block

Os elementos inline-block tem a largura definida por seu conteúdo, ficam um ao lado do outro e ele passa para baixo quando a linha é quebrada.
